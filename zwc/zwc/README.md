# ZWC

A mathematical computation library for array processing and numerical calculations.

## Installation

```bash
pip install zwc
```

## Quick Start

```python
import zwc
from zwc import rfx

# Create arrays
data = zwc.yitaf([1, 2, 3, 4, 5])
matrix = zwc.yitaf([[1, 2], [3, 4]])

# Mathematical operations
result = zwc.lahonig(data, 10)
mean_val = zwc.kocito(data)

# Specialized functions
inverse = rfx.quvohe(matrix)
solution = rfx.sarik(matrix, zwc.yitaf([5, 6]))
```
